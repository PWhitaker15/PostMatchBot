--INSTALLATION & SETUP--
1. Download Python https://www.python.org/downloads/
2. Open command prompt
3. Run $py -m pip install requests
4. Run $py -m pip install --upgrade Pillow
5. Change line in oldreplay.txt to path of Rocket League replays folder
6. Change line in token.txt to authentication token from Ballchasing
7. Change path of command prompt to folder with PostMatch.py
8. Run $py PostMatch.py
